## Key Documentation Features

### 1. Correct Block Structure Rules

All documentation emphasizes the critical EDS requirement:

✅ **Use ASCII tables** with `+`, `-`, `|` characters
❌ **Never use** standard markdown tables

Example documented throughout:
```
+--------------------------------------------------------------------------+
| Block Name                                                               |
+--------------------------------------------------------------------------+
| Content with backslash for line breaks\                                  |
| \                                                                        |
| More content                                                             |
+--------------------------------------------------------------------------+
```

- Square: 1024x1024
